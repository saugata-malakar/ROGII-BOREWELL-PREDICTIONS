What it does

A simple product story, not a code dump.

Reads well logs

It loads the well files, matches each horizontal well with its typewell companion, and creates a single consistent table per well.

Prepares usable features

It cleans missing values, creates depth and distance signals, and keeps only fields that the train and test sets truly share.

Returns a prediction file

It trains a model and writes the final TVT output in the exact submission format needed by Kaggle.

How it works

Walkthrough of the complete flow.

01

Upload data

The notebook finds the competition dataset automatically from the Kaggle environment.

02

Combine well files

Each horizontal file is paired with its typewell file to build one training table.

03

Engineer signals

Depth shifts, distance measures, and derived trends are added to improve prediction quality.

04

Predict TVT

LightGBM produces the final TVT values and exports them in the exact competition format.
